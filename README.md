# California House Price Prediction

This project predicts California house prices using Machine Learning and LightGBM.

## Features

- Data preprocessing
- Missing value handling
- One-hot encoding
- Feature scaling
- LightGBM Regression
- Model evaluation
- Streamlit deployment

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- LightGBM
- Streamlit

## Dataset

California Housing Dataset

## Model Performance

- Linear Regression Accuracy: ~62%
- LightGBM Accuracy: ~83%

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
streamlit run app.py
```
